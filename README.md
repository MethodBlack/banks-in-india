# Banks in India
A collection of websites & icons, and more.

## Installation
Install Banks in India from your terminal via npm.

```
npm i banks-in-india
```

## Banking Structure

```

                     ┌─────────────────────────┐
                     │   Ministry of Finance   │

                                  │

                        └─────────┬─────────┘
                                  │
                      ┌───────────┴───────────┐
                      │                       │
            ┌─────────▼─────────┐   ┌─────────▼─────────┐
            │  Scheduled Banks  │   │ Unscheduled Banks │
            └─────────┬─────────┘   └───────────────────┘
                      │
          ┌───────────┴───────────┐
          │                       │
┌─────────▼─────────┐   ┌─────────▼─────────┐
│ Commercial Banks  │   │Co-operative Banks ├─┬─▶Urban Co-operative
└───────────────────┘   └───────────────────┘ └─▶State Co-operative
          │
          │ ┌───────────────────────────┐
          ├─▶   Private Sector Banks    ├──────┬─▶Private Sector Banks
          │ └───────────────────────────┘      ├─▶Local Area Banks (LAB)
          │                                    ├─▶Small Finance Banks (SFB)
          │                                    └─▶Payments Banks (PB)
```
