Installation

Install from your terminal via npm.

npm i banks-in-india

Below is the directory structure:

.
└── banks-in-india/
    ├── datasets/
    │   └── banks-names.json
    └── icons/
        ├── financial_institutions/
        ├── foreign_banks/
        ├── local_area_banks/
        ├── payment_banks/
        ├── private_sector_banks/
        ├── public_sector_banks/
        ├── regional_rurual_banks/
